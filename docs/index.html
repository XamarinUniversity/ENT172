Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin.Forms development environment with either an emulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class, you will use the Xamarin.Auth component to access a web service protected by OAuth 2.0. The service is an ASP.NET website using OWIN and Katana - you can create the same basic service with the online tutorial if you'd like to control both sides.

Objectives

  1. Authorize with an OAuth 2.0 server
  2. Access an OAuth 2.0-secured API
  3. Exchange a Refresh Token for a new Access Token

Lecture

ENT172 - Accessing OAuth 2.0 Web Services with Xamarin.Auth.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Create an OAuth2Authenticator
    Exercise
    In this exercise, we will add the prerequisite components and the core code to authenticate using OAuth 2.0.
  2. Authenticate with an OAuth 2.0 protected web service
    Exercise
    In this exercise, we will authenticate a user against an OAuth 2.0 protected web service.
  3. Access data on an OAuth 2.0 protected service
    Exercise
    In this exercise, we will use an Access Token to retrieve data stored on a protected server.
  4. Use a Refresh Token to retrieve a new Access Token
    Exercise
    In this exercise, we will use a Refresh Token to retrieve a new Access Token when the previous Access Token expires.
